Implementation of [A Scalable and Production Ready Sky and atmosphere Rendering Technique](https://sebh.github.io/publications/egsr2020.pdf). Atmosphere perspective is not a part of this implementation.
Built and tested on Windows with msvc complier.

Features [Garlin's optimisation technique](https://fgarlin.com/blog/spectral-sky/) to replace linear rgb representation

Shadertoy implementations used as references to help integrate physical functions into code and adapted to Vulkan API:
https://www.shadertoy.com/view/fd2fWc
https://www.shadertoy.com/view/slyBDG
https://www.shadertoy.com/view/msXXDS
